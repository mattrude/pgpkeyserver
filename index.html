About this service

This web page is provided as a convenient alternative to the usual methods of interacting with the keyserver. Key submission and retrieval are usually handled by key management software (such as GnuPG) on your local system.

This keyserver supports both the traditional email interface pks (for key submission only), and the more common hkp interface, for interaction with PGP key management software.
